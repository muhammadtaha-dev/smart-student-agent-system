# Smart Student Profile Agent System

This is a beginner-level Agentic AI project built using Python.

## Features
- Add student details
- Calculate average marks
- Assign grades
- Intelligent status (Top Student / Needs Improvement)
- Search student by name
- Prevent duplicate entries using sets

## Concepts Used
- Python (Lists, Dictionaries, Sets)
- Functions
- Loops (for, while)
- Conditional Statements
- Match-case

## How to Run
1. Download the file
2. Run in Python:
   python student_agent_system.py
